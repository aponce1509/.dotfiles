\section{Schr\"odinger Equation}
The time-dependent Schr\"odinger equation is given by:
\begin{equation}
  i\hbar \frac{\partial}{\partial t} \Psi(\mathbf{r}, t) = \hat{H}
  \Psi(\mathbf{r}, t),
\end{equation}
where $\hat{H}$ is the Hamiltonian operator.
